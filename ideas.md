**Diseño: Cosmic Education - Futurismo Educativo Espacial**

**Movimiento de Diseño**: Inspirado en el Futurismo Digital y la estética Cyberpunk Educativa, fusionando elementos espaciales con tecnología educativa avanzada.

**Principios Fundamentales**:
1. **Profundidad Espacial**: Uso de capas visuales con parallax para crear sensación de inmersión en un universo educativo
2. **Tecnología Humanizada**: Equilibrio entre elementos tecnológicos (IA, robots) y calidez humana (profesor, testimonios)
3. **Energía Visual**: Uso estratégico de efectos de brillo (glow), partículas flotantes y animaciones sutiles que transmiten dinamismo
4. **Jerarquía Cromática**: Los colores no solo decoran, sino que guían la atención y comunican estados emocionales

**Filosofía de Color**:
- **Base Espacial**: Navy profundo (#0a1628) como lienzo que evoca el espacio infinito del conocimiento
- **Cyan Tecnológico**: (#00d4ff) representa la IA y la innovación, usado para elementos interactivos y tecnológicos
- **Dorado Aspiracional**: (#ffd700) simboliza el éxito académico y los logros, reservado para CTAs y badges importantes
- **Verde Crecimiento**: (#00ff88) indica progreso, aprobación y acciones positivas
- **Intención Emocional**: El contraste entre el fondo oscuro y los acentos brillantes crea un efecto de "luz en la oscuridad" que representa el conocimiento iluminando la confusión académica

**Paradigma de Layout**:
- **Asimetría Dinámica**: Evitar layouts centrados tradicionales; usar composiciones diagonales y en Z-pattern
- **Secciones Flotantes**: Tarjetas y contenedores con sombras profundas que parecen flotar sobre el fondo espacial
- **Grid Roto**: Romper la cuadrícula tradicional con elementos que se superponen y crean profundidad
- **Espaciado Respiratorio**: Uso generoso de whitespace (o "darkspace" en este caso) para que cada sección respire

**Elementos Distintivos**:
1. **Partículas Estelares Animadas**: Pequeñas estrellas y partículas que flotan sutilmente en el fondo, creando movimiento constante
2. **Bordes Neón**: Tarjetas con bordes de colores brillantes (cyan, verde, amarillo) que crean efecto de hologramas futuristas
3. **Personajes Ilustrados**: El profesor y el robot IA como elementos visuales recurrentes que humanizan la tecnología

**Filosofía de Interacción**:
- **Hover Elevación**: Los elementos interactivos se elevan con sombras más pronunciadas al hacer hover
- **Glow Responsivo**: Los botones y enlaces emiten un brillo sutil que se intensifica con la interacción
- **Transiciones Fluidas**: Todas las transiciones usan easing suave (ease-out) para sentirse naturales, no robóticas
- **Feedback Inmediato**: Cada acción del usuario recibe respuesta visual instantánea

**Sistema de Animación**:
- **Entrada Escalonada**: Los elementos entran en escena con fade-in + slide-up escalonado (stagger) al hacer scroll
- **Parallax Sutil**: El fondo se mueve a diferente velocidad que el contenido, creando profundidad
- **Pulse en CTAs**: Los botones principales tienen una animación de pulse muy sutil que atrae la atención sin ser molesta
- **Partículas Flotantes**: Elementos decorativos (estrellas, átomos) con movimiento perpetuo usando keyframes

**Sistema Tipográfico**:
- **Display (Títulos)**: 'Space Grotesk' - fuente geométrica con personalidad tech pero legible
- **Cuerpo**: 'Inter' - neutral, moderna, excelente legibilidad en pantalla
- **Acentos Numéricos**: 'JetBrains Mono' - monoespaciada para números, precios y datos técnicos
- **Jerarquía**: H1 (56px/bold), H2 (42px/bold), H3 (32px/semibold), Body (18px/regular), Small (14px/regular)
- **Peso Variable**: Alternar entre regular (400), medium (500), semibold (600) y bold (700) para crear ritmo visual

**Diseño: Organic Tech - Tecnología Cálida y Accesible**

**Movimiento de Diseño**: Inspirado en el Diseño Suizo moderno con influencias de Flat Design 2.0, priorizando la claridad y la accesibilidad sin sacrificar personalidad.

**Principios Fundamentales**:
1. **Claridad Absoluta**: Cada elemento tiene un propósito claro; cero decoración sin función
2. **Calidez Tecnológica**: Suavizar la frialdad de la tecnología con bordes redondeados y colores vibrantes
3. **Modularidad**: Sistema de componentes reutilizables que mantienen consistencia visual
4. **Accesibilidad Primero**: Contraste WCAG AAA, tamaños de fuente generosos, áreas de click amplias

**Filosofía de Color**:
- **Base Neutra Clara**: Blanco (#ffffff) y gris claro (#f5f7fa) para máxima legibilidad
- **Acentos Vibrantes**: Cyan (#00d4ff), Verde (#00ff88), Amarillo (#ffd700) usados con moderación
- **Navy Profundo**: (#0a1628) para texto y elementos de alto contraste
- **Intención Emocional**: Colores brillantes sobre fondo claro transmiten optimismo y energía juvenil

**Paradigma de Layout**:
- **Grid Modular**: Sistema de 12 columnas estricto pero con variaciones creativas
- **Secciones Alternadas**: Fondo blanco / gris claro alternando para separar contenido
- **Tarjetas Elevadas**: Cards con sombras suaves y bordes redondeados generosos (16-24px)
- **Contenido Centrado con Márgenes Amplios**: Max-width de 1200px con padding lateral generoso

**Elementos Distintivos**:
1. **Ilustraciones Planas**: Personajes y elementos en estilo flat con colores sólidos
2. **Iconografía Consistente**: Sistema de iconos outline con stroke de 2px
3. **Badges Coloridos**: Etiquetas con colores vibrantes para destacar información clave

**Filosofía de Interacción**:
- **Hover Sutil**: Cambio de color y elevación mínima (2-4px)
- **Transiciones Rápidas**: 200-300ms para mantener sensación de inmediatez
- **Estados Claros**: Cada estado interactivo es visualmente distinto
- **Feedback Visual**: Cambios de color instantáneos en elementos interactivos

**Sistema de Animación**:
- **Fade-in Simple**: Elementos aparecen con fade-in al scroll
- **Micro-interacciones**: Botones con scale sutil al click
- **Sin Parallax**: Movimiento mínimo para mantener claridad
- **Loading States**: Spinners y skeletons claros durante cargas

**Sistema Tipográfico**:
- **Todo**: 'Inter' - una sola familia tipográfica para máxima consistencia
- **Jerarquía por Peso**: Usar weights 400, 500, 600, 700, 800 para crear jerarquía
- **Tamaños Generosos**: H1 (48px), H2 (36px), H3 (28px), Body (16px)
- **Line-height Amplio**: 1.6 para body text, 1.2 para headings

4. **Maximalismo Controlado**: Mucha información visual pero organizada jerárquicamente

- **Diagonales Dominantes**: Secciones con cortes diagonales usando clip-path
- **Superposiciones Atrevidas**: Elementos que se superponen creando capas visuales

- **Márgenes Negativos**: Secciones que invaden el espacio de otras para crear continuidad
